NeoFermi Editor Help

Markdown with NeoFermi

Write markdown with embedded NeoFermi calculations. Use fenced code blocks for calculations:

```
x = 10 to 100 meters
y = x * 2
```

Inline Variables

Reference variables in text using ${'{'}varname{'}'}:

The distance is approximately ${'{'}x{'}'}.

Distribution Syntax

1 to 10 m            # Lognormal (68% CI)
1 .. 10 m            # Uniform distribution
100 +- 5 kg          # Normal (mean +/- sigma)
3 of 10              # Beta (3 successes, 10 trials)
100 * 10%            # Twiddle by ±10%
{365: 303, 366: 97}  # Weighted set

Scale-word ranges

1 to 2 million       # 1e6 to 2e6
1 .. 2 billion USD   # Uniform 1e9 to 2e9 USD
1 +/- 2 million      # Normal mean 1e6, sigma 2e6

Unit Conversion

100 meters as feet       # Convert to specific unit
x -> km                  # Alternative syntax
98.6 degF as degC        # Affine temperature
1 feet / 1 mm as feet/mm # Keep ratio in compound form
100 miles as SI          # Convert to SI base units

Dates & Durations

#2026-04-16#                 # Date literal
#2026-04-16T12:30#           # With time
#2027-01-01# - #2026-01-01#  # Duration (365 day)

Units & Constants

m, kg, s, A, K       # SI base units
calorie, kcal, Cal   # Energy (Cal = kcal)
parsec, ly, barn     # Astronomy / nuclear
knot, mph, atm, hp   # Misc

USD, EUR, GBP, JPY   # Currencies (12 supported)

c, h, hbar, G, k     # Physics constants
M_earth, R_earth     # Earth
world_population     # Demographics

LaTeX Math

Write LaTeX equations inline with $...$ or as display blocks with $$...$$:

Inline: $E = mc^2$
Display: $$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$

Standard LaTeX environments are also supported:

\begin{equation}
  F = G \frac{m_1 m_2}{r^2}
\end{equation}

Keyboard Shortcuts

Ctrl/Cmd+S           # Share (copy link)
Ctrl/Cmd+E           # Export as markdown
